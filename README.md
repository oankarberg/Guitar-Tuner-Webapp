# Guitar-Tuner-Webapp
Guitar Tuner App in TFYA65

A guitar tuner web app using webRTC to enable real time communication with the user through the microphone. Have `npm` installed and `bower` installed to run the app.

```
npm install
```
```
bower install
```
``` 
grunt serve
```
